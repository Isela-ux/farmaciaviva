# Evidencia — Farmacia Viva VIC 2026

Carpeta para capturas de pantalla y fotos de sesiones con el asesor.

## Estructura

```
evidencia/
├── MAPEO_CAPTURAS.md      ← qué guardar y qué ya tienes
├── semana4/
│   ├── alumno-a/          ← A1.png … A15.jpg
│   └── alumno-b/          ← B1.png … B13.jpg
└── semana5/
    ├── alumno-a/          ← S5-A1 … S5-A6
    └── alumno-b/          ← S5-B1 … S5-B7
```

## Cómo usar

1. Lee [`MAPEO_CAPTURAS.md`](./MAPEO_CAPTURAS.md)  
2. Renombra y guarda tus capturas existentes en la carpeta correcta  
3. Toma las que falten (checklist en cada `CHECKLIST.md`)  
4. Adjunta la carpeta `evidencia/` al entregar o súbela a Drive y enlaza en el reporte
